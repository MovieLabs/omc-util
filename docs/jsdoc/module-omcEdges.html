(static) edgeCreate(params)

Description:

  • Creates a new edge from one entity to another, based on the allowed edges for the entity

    • Setting the 'inverse' property will also create the inverse edge in the toEntity if applicable
    • Some entities have multiple properties where the same toEntity is allowed, using the intrinsicEdge property allows a specific one to be specified
Source:
  • src/omc/omcEdges.mjs, line 226
Parameters:
Name Type Description
params Object
Properties
Name Type Description
fromEntity OmcEntity

The entity on which to create the new edge
toEntity OmcEntity

The entity to which the edge should be created
forEntity OmcEntity

Use when setting edges in a Context, this is the entity for which the Context is For.
intrinsicEdge Object

Specify a specific edge, for entities that have multiple valid edge patterns, this denotes the specific one to use
inverse boolean

Whether the inverse edge should also be set if there is one
edgeType 'edges' | 'intrinsic'

Which edge table to use, intrinsic or regular edges.
